## Notes Structure

Each note contains the following elements:

- Title (#)
- Tags (##)
  - Tags are written in `kebab-case` with the prefix `tag-` to be easy to be searched.
- Notes (##)
- Backlinks (##)
  - Refer to notes
- References (##)
